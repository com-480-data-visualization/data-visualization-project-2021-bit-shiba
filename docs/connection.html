There are plenty kinds of mix-type Pokémon that make up the wonderful Pokémon world. We use the chord map to help to get an overview of how a mix-type Pokémon looks like and their abilities.

There are around 129 mix-types Pokémon there, When the mouse hovers over the link, it will tell the number so that we can easily get which type of Pokémon has the most and which mix-type is unusual. The water has the largest family which reach almost 136kinds and second comes to flying. The ice type Pokémon has the lowest number only 42, even less than the dragon type. Surprisingly, the fly type Pokémon usually have two type, pure fly type Pokémon has only three kind. And the bug as the second type also very rare.
